SESSION 2 — BUILDING FLOW
VANCE (INTRO)

This is Session Two.

You’ve learned how to stand.
You’ve learned how to protect yourself.
You’ve thrown your first punches.

Today is about connection.

Not power.
Not speed.

Flow.

Here is Mac.

[4s MUSIC GAP — VANCE → MAC]

MAC (INTRO)

Mac here.
Session Two.

Today we reinforce what you learned.

Same stance.
Same guard.
Same punches.

But now we start to let them flow together.

Nothing new yet.
Just cleaner movement.

MAC (STANCE CHECK — CONTEXT)

Set your stance.

Train tracks.
Not a tightrope.

Front foot forward.
Back foot turned slightly.

Knees soft.
Weight balanced.

This is your base.
We always come back here.

Hold it.

MAC (GUARD RESET — CONTEXT)

Hands up.

Remember — this isn’t tension.
It’s readiness.

When the guard stays up,
you stay aware.

Set the hands.
Leave them there.

After me.
Set of four.

Hands set. One.
Hands set. Two.
Hands set. Three.
Hands set. Four.

Again.

One.
Two.
Three.
Four.

Last set.

One.
Two.
Three.
Four.

And stop. Stay in stance.

Back to the jab.

This punch controls space.
It keeps things where you want them.

Loose arm.
Fast out.
Fast back.

Wet towel.
Snap, then reset.

Set of four.

Jab. One.
Jab. Two.

Jab. Four.

Again.

Two.
Three.
Four.

Last set.

One.

[2s MUSIC GAP — JAB → CROSS REVIEW]

MAC (CROSS REVIEW — CONTEXT)

Now the cross.

This is where the body helps the arm.

Push the floor.
Turn the hip.
Let the punch follow.

Heavy door.
You push it shut — you don’t slam it.

After me.
Set of four.

Now we connect them again.

Jab sets the line.
Cross travels through it.

This combination matters.
You’ll come back to it again and again.

Later, I’ll call this one–two.
For now, I’ll still say the punches.

After me.
